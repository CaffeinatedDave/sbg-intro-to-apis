GET /intro/hello

Hello world response

Query params: name - string
Replaces "world" in the greeting response

POST /intro/echo

echo service - replays any request given as a response

GET /intro/random/:number

random number generator

Params: number - integer
num must be greater than 4.
Returns a random dice roll for a 'number'-sided dice.

GET /intro/random/stats

returns statistics on the random number generator

GET /v2/things

Returns the JSON representation of the current list

POST /v2/things

Adds an item to "the list"

Param: thing - string
The thing to be added to the list

Returns the JSON representation of the current list

DELETE /v2/things

Deletes from the list

Param: thing - string
The thing to be deleted from the list

Returns the JSON representation of the current list
